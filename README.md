# Chatify - Real-Time Chat Application

A full-stack **real-time chat application** built with **React (Vite)** frontend, **Node.js + Express + Socket.IO** backend, and **MongoDB** database.

---

## Features

- Real-time chat using **Socket.IO**
- User authentication with **JWT** and cookies
- Signup/Login with validation
- Profile update
- Online users list
- Mobile and desktop friendly
- Responsive UI with **TailwindCSS** + **DaisyUI**

---

## Tech Stack

**Frontend:**

- React 19 + Vite
- TailwindCSS + DaisyUI
- Zustand for state management
- Axios for API calls
- React Router v7
- Socket.IO Client
- React Hot Toast

**Backend:**

- Node.js + Express
- MongoDB
- Socket.IO Server
- JWT Authentication
- Cookie-based auth
- CORS enabled for cross-origin
